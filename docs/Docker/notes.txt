5. Ryd op i images + system

👉 fjerner unused resources

6. Docker build cache optimering
Dårlig Dockerfile:
COPY . .
RUN mvn package
God Dockerfile (cache-optimeret):
COPY pom.xml .
RUN mvn dependency:go-offline

COPY src ./src
RUN mvn package

👉 hvorfor det virker:

dependencies cached
src ændringer invaliderer ikke alt
